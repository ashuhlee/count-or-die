## Core Gameplay
- Increase your count to match the current goal before the timer expires
- Each goal reached increases the timer speed
- Can't click fast enough? Boosts can jump you to the goal (up to 4 times per game)

## Build from Source
For local development or contributing:
1. Clone this repository:
```zsh
git clone https://github.com/ashuhlee/count-or-die.git
cd count-or-die
```
2. Install dependencies:
```zsh
npm install
```
3. Run the app in development mode:
```zsh
npm start
```
4. Package the app and build distributables (optional):
```bash
npm run package
npm run make
```
